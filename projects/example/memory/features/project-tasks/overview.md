# Feature: project tasks

Example feature memory for the flagship sample in `sample/project-tasks-app/`.

## What it is

A SaaS-style workflow for creating projects, adding tasks, assigning tasks to workspace members, enforcing due dates, changing task status, and recording audit logs.
